# AtmUygulamas-
Bu proje, JavaScript kullanılarak geliştirilmiş basit bir ATM simülasyonudur. Kullanıcıların para çekme, para yatırma ve bakiye görüntüleme işlemlerini gerçekleştirmesini sağlar. Temel finansal işlem mantığını ve kullanıcı girdisi kontrolünü uygulamaya odaklanır.
